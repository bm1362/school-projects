Plot 3D (v1.0)

My first JS app, written in December 2011. Plot 3D is a graphing tool to render 3 variable functions using Canvas and Three.js. Takes input in the form of f(x,y) = z.

Input Examples for Plot3D:
x2 = x*x,
2x = 2 * x,
Sin(x) = Math.sin(x), Math.cos(x), Math.sin(x), Math.tan(x) etc.
Only compatible with Google Chrome, Firefox 4 Beta

Graffiti (v1.0)

An example of an event logging library I wrote in JavaScript. It allows the user to toggle on and off a canvas overlay- features a fully functioning drawing app. The user can load iframes or swfobjects into the drawing area.
Work in progress.

Nodes (v1.0)

A JavaScript implementation of Dijkstra's Algorithm used to find the minimal distance from node A to any other node in the network. Randomly places nodes and creates connections. I have no interest in pursuing this further, wrote it one morning. I'm going to work on an A* path finding algorithm next.
